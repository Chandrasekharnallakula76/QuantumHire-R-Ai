CognitiveScreen AI — Embed Guide

Drop the AI interview platform into any website, LMS, or HR portal with a single script tag or iframe.

Recommended

Method 1 — Script Tag (Auto-Init)

Paste this into your HTML. The widget auto-initializes from data attributes. Replace YOUR_FRONTEND_URL with your deployed frontend URL.

<!-- CognitiveScreen AI Interview Widget -->
<div id="cs-interview"></div>

<script src="https://YOUR_FRONTEND_URL/embed.js"
  data-container="#cs-interview"
  data-email="candidate@example.com"
  data-name="John Doe"
  data-role="Sales"
  data-height="100vh">
</script>

Method 2 — JavaScript API

Full programmatic control with callbacks for interview completion and vitals data.

<div id="interview-box"></div>
<script src="https://YOUR_FRONTEND_URL/embed.js"></script>

<script>
  CognitiveScreen.init({
    container: '#interview-box',
    email:  'candidate@company.com',
    name:   'Jane Smith',
    role:   'Project Management',
    width:  '100%',
    height: '700px',

    // Called when interview finishes
    onComplete: function(result) {
      console.log('Score:', result.total_score);
      console.log('Result:', result.result);
    },

    // Called with real-time vitals data
    onVitals: function(vitals) {
      console.log('Heart Rate:', vitals.hr);
    }
  });
</script>

Method 3 — Raw iframe

If you can't load external scripts, use a plain iframe. Camera/mic permissions are required.

<iframe
  src="https://YOUR_FRONTEND_URL/#/embed?email=candidate@example.com&name=John+Doe&role=Sales"
  width="100%"
  height="700"
  style="border:none; border-radius:12px;"
  allow="camera; microphone; display-capture; autoplay; fullscreen"
  allowfullscreen
></iframe>
Important: The allow="camera; microphone" attribute is required for the webcam vitals scan and voice recording to work inside the iframe.

Configuration Parameters

ParameterTypeDefaultDescription
containerstring | Element—CSS selector or DOM element to mount the widget into
emailstring""Pre-fill candidate email on the OTP screen
namestring""Candidate name (shown in face HUD and results)
rolestring"General"Interview role category for question selection
widthstring"100%"Widget width (CSS value)
heightstring"100vh"Widget height (CSS value)
themestring"dark"Color theme: dark or light
onCompletefunction—Callback when interview finishes (receives result object)
onVitalsfunction—Callback for real-time vitals updates
